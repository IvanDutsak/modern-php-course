# Проект 2: Браузер даних про якість повітря

## Опис проекту

Цей проєкт призначений для відображення, аналізу та візуального представлення показників якості повітря у різних містах. Він збирає структуровані дані про індекс якості повітря (AQI) і надає зрозумілий інтерфейс для перегляду поточних значень, моніторингу змін у часі та порівняння локацій.

## Технології

- PHP
- JSON
- HTML/CSS
- Робота з файлами
- Парсинг даних

## Основні функції

1. **Перегляд міст** - відображення списку міст з показниками AQI
2. **Детальна інформація** - перегляд детальних даних про якість повітря
3. **Кольорові індикатори** - візуальне представлення рівня забруднення
4. **Історичні дані** - перегляд змін показників у часі

## Структура проекту

```
src/
├── index.php            # Головна сторінка з переліком міст
├── city.php             # Детальна інформація про місто
├── data/
│   └── index.json       # Дані про якість повітря
├── styles/
│   └── simple.css       # Стилі
└── views/
    ├── header.php       # Шапка
    └── footer.php       # Підвал
```

## Як запустити

1. Розмістіть файли в директорії веб-сервера
2. Переконайтеся, що файл `data/index.json` доступний для читання
3. Відкрийте `http://localhost/index.php`

## Ключові концепції

### Робота з JSON

```php
$jsonData = file_get_contents('data/index.json');
$data = json_decode($jsonData, true);
```

### Кольорові індикатори AQI

```php
function getAQIColor($aqi) {
    if ($aqi <= 50) return 'green';
    if ($aqi <= 100) return 'yellow';
    if ($aqi <= 150) return 'orange';
    return 'red';
}
```

## Рівні якості повітря

| AQI | Рівень | Колір |
|-----|--------|-------|
| 0-50 | Добрий | Зелений |
| 51-100 | Помірний | Жовтий |
| 101-150 | Нездоровий для чутливих груп | Помаранчевий |
| 151+ | Нездоровий | Червоний |
