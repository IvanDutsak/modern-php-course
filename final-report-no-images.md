## Теоретичні відомості

У PHP масиви є універсальною структурою даних, яка дозволяє зберігати впорядковані та невпорядковані дані. Масиви підтримують індексовану та асоціативну форму, а також можуть містити вкладені масиви, що робить їх схожими на об’єкти в JavaScript. Для роботи з масивами використовують функції `in_array`, `sort`, `array_search`, `array_slice`, `array_sum`, `array_merge` і цикл `foreach`. У вкладених структурах дані перебираються через кілька рівнів вкладеності, що дозволяє працювати зі складними наборами інформації.

PHP дозволяє приймати параметри з адресного рядка через `$_GET`, працювати з формами і підключати зовнішні файли за допомогою `include`, який виконується прямо у місці виклику. Для роботи з файлами використовують `file_get_contents()`, а для аналізу текстових даних - функції `str_starts_with()` і `str_ends_with()`. Завдяки цим можливостям PHP може взаємодіяти і з клієнтом (форми), і з серверними ресурсами (файли, URL).

Сучасний спосіб роботи з базою у PHP - через PDO. Підключення виконується через об’єкт PDO, після чого можна використовувати підготовлені запити (`prepare`), підставляти значення (`bindValue`) та отримувати результати (`fetchAll`). Обробка помилок реалізується через `try-catch` та виняток `PDOException`, що дозволяє безпечно виконувати запити та керувати помилками SQL.

PHP підтримує повноцінне ООП: класи, об’єкти, наслідування (`extends`), інтерфейси (`implements`) та абстракції. Простори імен (`namespace`) забезпечують впорядкування структури проєкту, а `use` дозволяє імпортувати класи. Магічні методи (`__construct`, `__toString`, `__get`, `__set`) дозволяють автоматизувати поведінку об’єктів. Поширений патерн — Singleton, який гарантує єдиний екземпляр класу.

## Проект 1: Багатосторінкова галерея зображень

**Опис:**
Це веб-застосунок на PHP, розроблений для створення інтерактивної фотогалереї з можливістю переглядання колекції зображень. Проект демонструє основні концепції веб-програмування на PHP, включаючи роботу з GET-параметрами, масивами даних та обробку URL-кодування для безпеки.

**GitHub:** [project-01-image-gallery](https://github.com/IvanDutsak/modern-php-course/tree/main/project-01-image-gallery)

**Демонстрація виконання:**

*   **Структура проекту:**

    ```
    src/
    ├── gallery.php
    ├── image.php
    ├── images/
    ├── inc/
    │   └── images.inc.php
    ├── styles/
    │   └── simple.css
    └── views/
        ├── header.php
        └── footer.php
    ```

*   **Ключовий код (gallery.php):**

    ```php
    <div class="gallery-container">
        <?php foreach($imageTitles AS $src => $title): ?>
        <a href="image.php?<?php echo http_build_query([\"image\" => $src]); ?>" class="gallery-item">
            <h3><?php echo e($title); ?></h3>
            <img src="./images/<?php echo rawurlencode($src); ?>" alt="<?php echo e($title); ?>" />
        </a>
    <?php endforeach; ?>
    </div>
    ```

**Висновок:**
Проект реалізує централізоване управління зображеннями через асоціативні масиви PHP, що дозволяє легко додавати нові фото. `gallery.php` показує зображення у сітці, а `image.php` - детальний перегляд із перевіркою параметрів та захистом від помилок. Використовуються функції `e()`, `rawurlencode()` та `rawurldecode()` для безпеки та коректної роботи з URL. Структура проекту розділяє логіку, шаблони та стилі, а `header.php` та `footer.php` забезпечують єдиний дизайн. Проект легкий для підтримки та масштабування.

## Проект 6: Дослідник міст

**Опис:**
Веб-додаток на PHP, розроблений для перегляду та управління інформацією про міста по всьому світу. Проект демонструє сучасні практики розробки на PHP, включаючи об\\'єктно-орієнтоване програмування та шаблон Repository.

**GitHub:** [project-06-city-explorer](https://github.com/IvanDutsak/modern-php-course/tree/main/project-06-city-explorer)

**Демонстрація виконання:**

*   **Repository Pattern:**

    ```php
    class CityRepository {
        private PDO $pdo;
        
        public function findById(int $id): ?City {
            $stmt = $this->pdo->prepare(\"SELECT * FROM cities WHERE id = ?\");
            $stmt->execute([$id]);
            $stmt->setFetchMode(PDO::FETCH_CLASS, City::class);
            return $stmt->fetch() ?: null;
        }
    }
    ```

**Висновок:**
Додаток реалізує повноцінну систему перегляду, деталізації та редагування міст із надійною пагінацією. Використання ISO-кодів і прапорів значно покращує зручність інтерфейсу. Безпечна робота з базою даних робить систему стабільною та масштабованою.

## Проект 8: Система керування контентом (CMS)

**Опис:**
Це мікро-CMS система на PHP з розділенням на адміністративну і публічну частини. Проект реалізований з використанням принципів MVC, IoC контейнера та репозиторіїв для роботи з даними.

**GitHub:** [project-08-cms-system](https://github.com/IvanDutsak/modern-php-course/tree/main/project-08-cms-system)

**Демонстрація виконання:**

*   **IoC Container:**

    ```php
    class Container {
        private array $recipes = [];
        
        public function bind(string $what, \Closure $recipe) {
            $this->recipes[$what] = $recipe;
        }
        
        public function get($what) {
            return $this->recipes[$what]();
        }
    }
    ```

*   **MVC Routing:**

    ```php
    match($route) {
        \"home\" => $container->get(\"pagesController\")->index(),
        \"admin/pages\" => $container->get(\"pagesAdminController\")->index(),
        default => $container->get(\"notFoundController\")->show()
    };
    ```

**Висновок:**
Проект реалізовано за принципами модульності та розподілу відповідальностей. Використання PDO, prepared statements, CSRF-захисту та перевірки сесій забезпечує високу безпеку і стабільність адміністрування. Архітектура з IoC контейнером та репозиторіями робить систему гнучкою, масштабованою та легкою для підтримки.
