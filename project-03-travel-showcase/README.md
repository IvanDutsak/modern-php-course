# Проект 3: Автоматична вітрина подорожей

## Опис проекту

PHP-проект, що автоматично оновлює вітрину подорожей на основі файлів у системі. Додаток самостійно зчитує зображення та описи з директорій та формує актуальну галерею без ручних змін у коді. Проєкт знайомить із роботою з файловою системою PHP, динамічним завантаженням контенту та побудовою гнучкої структури веб-додатків.

## Технології

- PHP
- Робота з файловою системою
- `scandir()`, `file_get_contents()`
- Динамічне завантаження контенту
- HTML/CSS

## Основні функції

1. **Автоматичне сканування** - зчитування файлів з директорії
2. **Динамічна галерея** - формування галереї без ручного кодування
3. **Читання описів** - завантаження текстових описів з файлів
4. **Гнучка структура** - легке додавання нових елементів

## Структура проекту

```
src/
├── index.php            # Головна сторінка
├── images/              # Папка з зображеннями та описами
│   ├── IMG_0933.txt     # Опис зображення
│   ├── IMG_3280.txt
│   └── ...
└── styles/
    └── simple.css       # Стилі
```

## Як запустити

1. Розмістіть файли в директорії веб-сервера
2. Додайте зображення та текстові файли з описами в папку `images/`
3. Відкрийте `http://localhost/index.php`

## Ключові концепції

### Сканування директорії

```php
$files = scandir('images/');
$files = array_diff($files, ['.', '..']);
```

### Читання вмісту файлу

```php
$description = file_get_contents("images/$filename.txt");
```

### Фільтрація файлів

```php
$imageFiles = array_filter($files, function($file) {
    return str_ends_with($file, '.jpg') || str_ends_with($file, '.png');
});
```
