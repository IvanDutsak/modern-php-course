## Про курс

Цей репозиторій містить практичні проекти, виконані в рамках курсу Modern PHP на платформі Udemy. Курс охоплює сучасні підходи до розробки на PHP, включаючи:

- Основи PHP та динамічні веб-сайти
- Робота з масивами та циклами
- URL параметри та форми
- Робота з базами даних (MySQL/MariaDB через PDO)
- Об'єктно-орієнтоване програмування (OOP)
- MVC архітектура
- Автентифікація та безпека
- Сучасні патерни проектування

## Структура проектів

Репозиторій організований за розділами курсу:

1. **Project 1: Image Gallery** - Багатосторінкова галерея зображень
2. **Project 2: Air Quality Browser** - Браузер даних про якість повітря
3. **Project 3: Travel Showcase** - Автоматична вітрина подорожей
4. **Project 4: Diary App** - Додаток-щоденник з базою даних
5. **Project 5: Name Explorer** - Дослідник імен з пагінацією
6. **Project 6: City Explorer** - Дослідник міст (OOP, Repository Pattern)
7. **Project 7: Weather App** - Додаток погоди (Strategy Pattern)
8. **Project 8: CMS System** - Система керування контентом (MVC, Authentication)

## Технології

- PHP 8.x
- MySQL/MariaDB
- PDO для роботи з БД
- HTML5/CSS3
- Object-Oriented Programming
- MVC Architecture
- Design Patterns
