# Проект 1: Багатосторінкова галерея зображень

## Опис проекту

Це веб-застосунок на PHP, розроблений для створення інтерактивної фотогалереї з можливістю переглядання колекції зображень. Проект демонструє основні концепції веб-програмування на PHP, включаючи роботу з GET-параметрами, масивами даних та обробку URL-кодування для безпеки.

## Технології

- PHP
- HTML/CSS
- GET параметри
- Асоціативні масиви
- URL encoding/decoding

## Основні функції

1. **Перегляд галереї** - відображення всіх зображень у вигляді сітки
2. **Детальний перегляд** - перегляд окремого зображення з описом
3. **Безпечна обробка URL** - використання `rawurlencode()` та `rawurldecode()`
4. **Динамічне завантаження** - дані зберігаються в масивах PHP

## Структура проекту

```
src/
├── gallery.php          # Головна сторінка галереї
├── image.php            # Сторінка детального перегляду
├── images/              # Папка з зображеннями
├── inc/
│   └── images.inc.php   # Дані про зображення
├── styles/
│   └── simple.css       # Стилі
└── views/
    ├── header.php       # Шапка сайту
    └── footer.php       # Підвал сайту
```

## Як запустити

1. Розмістіть файли в директорії веб-сервера (наприклад, `htdocs` для XAMPP)
2. Відкрийте браузер та перейдіть до `http://localhost/gallery.php`
3. Натисніть на будь-яке зображення для детального перегляду

## Ключові концепції

### Робота з GET-параметрами

```php
$imageId = $_GET['image'] ?? null;
```

### Безпечне кодування URL

```php
$encodedImage = rawurlencode($imageId);
```

### Асоціативні масиви

```php
$imageTitles = [
    'IMG_0933.jpg' => 'Mountain View',
    'IMG_3280.jpg' => 'Sunset Beach'
];
```
